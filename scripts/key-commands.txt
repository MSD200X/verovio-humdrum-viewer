
<b> Navigation commands <b>

?	toggle this help menu
&larr;/&rarr;	next/previous page
<i>home</i>	go to first page
<i>end</i>	go to last page
<i>space</i>	start/stop score playback
,	toggle CSV/TSV display of Humdrum data
< >	+/&ndash; tab size in editor
<nobr>shft-&larr;/&rarr;</nobr>	next/previous work
shft-up	go back to repertory index

<b> More commands (+<i>alt</i> key) <b>
b	show/hide logo
f	(un)freeze dynamic notation
g	display SVG text for current page
h	show Humdrum data
m	show MEI data
o	toggle original clefs
p	show source scan (when available)
r	reload file from server
s	save editor contents in local file
v	toggle vi/plain editor
y	show/hide data editor

<p><b> Note editing commands </b></p>

a	force stem up
b	force stem down
c	clear stem direction
i	toggle editorial accidental
m/M	toggle lower mordent
n	toggle natural on note
p	toggle pedal start before note
P	toggle pedal stop after note
q	toggle grace-note style
t/T	toggle trill
w/W	toggle upper mordent
x	force display of accidental
#	toggle sharp on note
'	toggle staccato
-	toggle flat on note
;	toggle fermata
^	toggle accent
shift-^	toggle marcato
`	toggle staccatissimo
~	toggle tenuto

<p><b> Rest editing commands </b></p>

y	make rest invisible

<p><b> Slur editing commands </b></p>

a	force slur above
b	force slur below
c	clear slur orientation
D	delete slur
&larr;	move slur start 1 note left
&rarr;	slur start 1 note right
#&larr;	slur start # notes left
#&rarr;	slur start # notes right
<nobr>shft-&larr;</nobr>	slur end left
<nobr>shft-&rarr;</nobr>	slur end right
<nobr># shft-&larr;</nobr>	slur end # notes left
<nobr># shft-&rarr;</nobr>	slur end # notes right

<p><b> Tie editing commands </b></p>

a	force tie above
b	force tie below
c	clear tie orientation

<p><b> Beam editing commands </b></p>

a	force beam above
b	force beam below
c	clear beam orientation
